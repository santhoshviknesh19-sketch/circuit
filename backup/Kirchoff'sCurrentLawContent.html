Kirchoff's Current Law Course

Master the fundamental concepts of Kirchoff's Current Law and its applications in circuit analysis.

What is Kirchoff's Current Law?

Kirchoff's Current Law (KCL) states that the algebraic sum of all currents entering and leaving a node in an electrical circuit is zero. This law is based on the principle of conservation of charge.

In mathematical terms, for any node in a circuit:

ΣI_in = ΣI_out

or

ΣI = 0

Figure 1: Node with three branches demonstrating Kirchoff's Current Law

The Mathematical Formula

KCL can be expressed as:

I₁ + I₂ + I₃ + ... + Iₙ = 0

Where each current is positive for currents entering the node and negative for currents leaving (or vice versa, depending on the chosen sign convention).

  • I is current in amperes (A)
  • Node is a junction where two or more circuit elements meet

Using KCL in Circuit Analysis

1. Identify all nodes in the circuit.

2. Choose a reference node (ground).

3. Apply KCL at each non-reference node: sum currents leaving the node = 0.

4. Solve the resulting equations to find unknown currents or voltages.

Example

For a node with three connected branches carrying currents I₁, I₂, and I₃ (entering):

I₁ + I₂ + I₃ = 0

If I₁ = 2A, I₂ = -1A (leaving), then I₃ = -1A (leaving) to satisfy KCL.

Practical Applications

KCL is widely used in:

  • Node-voltage analysis
  • Designing and analyzing transistor amplifiers
  • Power distribution networks
  • SPICE-based circuit simulation
